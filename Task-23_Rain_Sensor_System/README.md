# Task 23 - Rain Sensor System

## Objective
To detect rain conditions using a rain sensor and indicate the detection using an LED.

## Components Required
- Arduino
- Rain Sensor
- LED
- 220Ω Resistor


## Working Principle
The rain sensor provides an analog value based on the amount of water detected. When the value exceeds a predefined threshold, the LED turns ON indicating rain.

## Result
Rain detection was successfully implemented using Arduino.
