# Task 22 - Soil Moisture Monitoring

## Objective
To monitor soil moisture levels using an analog sensor and arduino

## Components Required
- Arduino
- Soil Moisture Sensor
- Jumper Wires



## Working
The Pico reads the analog value from the soil moisture sensor and displays the moisture level through Serial Output.

## Output
Moisture Value: 0 - 855

## Result
Successfully monitored soil moisture levels using arduino
