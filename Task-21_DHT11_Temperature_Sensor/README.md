# Task 21 - DHT11 Temperature Sensor

## Objective
To measure temperature and humidity using a DHT11 sensor and display the values on the Serial Monitor.

## Components Required
- Arduino UNO
- DHT11 Temperature and Humidity Sensor
- Jumper Wires

## Connections

| DHT11 Pin | Arduino UNO |
|------------|-------------|
| VCC | 5V |
| GND | GND |
| DATA | D2 |

## Working Principle
The DHT11 sensor measures temperature and humidity from the surrounding environment and sends the data to the Arduino. The Arduino reads the sensor values and displays them on the Serial Monitor.

## Output
- Temperature in °C
- Humidity in %

## Simulation
Wokwi Simulator

## Result
Temperature and humidity values were successfully measured and displayed on the Serial Monitor.
